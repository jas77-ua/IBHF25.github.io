# Sabadell First

> App bancaria innovadora diseñada específicamente para jóvenes de 14-17 años

## Descripción

**Sabadell First** es la primera aplicación bancaria creada por Banco Sabadell para adolescentes, combinando educación financiera, gamificación y control parental en una experiencia moderna y segura.

### Problemas que Resuelve

- **Falta de acceso bancario para menores**: Primera cuenta bancaria supervisada para jóvenes
- **Captación temprana de clientes**: Los usuarios crecen con Sabadell y permanecen a largo plazo
- **Educación financiera deficiente**: Introduce conceptos de ahorro, inversión y gasto responsable desde temprana edad
- **Desmitificación bancaria**: Elimina prejuicios sobre entidades bancarias mediante una experiencia transparente y amigable

### Público Objetivo

- Adolescentes entre **14 y 17 años**
- Requiere autorización y supervisión de padres/tutores legales
- Residentes en España

### Cuota de Mercado Potencial

- **+2.5 millones** de jóvenes entre 14 y 17 años en España (2025)

## Funcionalidades Implementadas

### 1. Autenticación y Seguridad

- **Pantalla Splash**: Animación de bienvenida con logo y barra de carga
- **Login Seguro**: Autenticación con DNI/NIE y contraseña
- **Diseño Juvenil**: Interfaz moderna y atractiva para el público objetivo

### 2. Dashboard Principal

- **Vista de Saldo**: Visualización clara del saldo disponible (ej: 453.30€)
- **Meta de Ahorro**: Seguimiento de objetivos de ahorro (ej: Meta 500€)
- **Barra de Progreso**: Indicador visual del progreso de ahorro (91%)
- **Tarjetas Bancarias**:
  - Tarjeta Débito Visa (termina en 4321)
  - Tarjeta Prepago Visa (termina en 1234)
- **Misiones Activas**: Vista rápida de misiones en curso
- **Últimas Transacciones**: Historial de movimientos recientes

### 3. Gestión de Tarjetas

- **Vista Detallada**: Tarjetas con diseño visual atractivo
- **Controles de Seguridad**:
  - Bloquear/Desbloquear tarjeta instantáneamente
  - Ver historial de transacciones por tarjeta
  - Gestionar límites de gasto
- **Información de Tarjeta**: Número parcial, vencimiento (12/27)

### 4. Cuentas Bancarias

- **Gestión de Cuentas**: Vista de todas las cuentas del usuario
- **Detalles de Cuenta**: Saldo, movimientos e información completa
- **Historial de Transacciones**: Registro detallado con filtros

### 5. Touch & Share - División de Gastos

Funcionalidad innovadora para compartir gastos entre amigos:

- **Configuración de Gasto**: Monto total y número de personas
- **Cálculo Automático**: Reparto equitativo instantáneo
- **Estado de Deudas**: Seguimiento de quién debe a quién
- **Pago NFC Simulado**: Funcionalidad de pago por aproximación
- **Historial de Pagos Compartidos**: Registro de divisiones anteriores

### 6. Sistema de Misiones Gamificado

Sistema de recompensas educativas con 6 misiones disponibles:

1. **Primer Ahorro Épico**
   - Objetivo: Ahorrar 50€ este mes
   - Recompensas: +5€ bonus + Insignia

2. **Gamer Legendario**
   - Objetivo: Completar 10 horas de estudio
   - Recompensas: Tarjeta Steam 20€ + Insignia Gamer

3. **Experiencia Musical**
   - Objetivo: Ayudar en tareas domésticas 15 días seguidos
   - Recompensas: Sorteo entradas concierto + Playlist exclusiva

4. **Aventura Cinéfila**
   - Objetivo: Leer 2 libros este mes
   - Recompensas: Entrada de cine + palomitas + Insignia Lector

5. **Chef en Formación**
   - Objetivo: Preparar 5 cenas para la familia
   - Recompensas: Vale 30€ restaurante + Delantal personalizado

6. **Deportista Épico**
   - Objetivo: Ejercitarse 3 veces por semana durante un mes
   - Recompensas: Ropa deportiva 40€ + Insignia Atleta

**Características del Sistema de Misiones**:
- Barra de progreso individual
- Filtros por categoría (Todas, Ahorro, Educación, Recompensas)
- Estadísticas de misiones completadas y recompensas ganadas

### 7. Transferencias y Pagos

Sistema completo de transferencias con límites de seguridad para menores:

- **Entre Mis Cuentas**: Transferencias instantáneas entre cuentas propias
- **Bizum**: Envío rápido con móvil (límite 50€)
- **Transferencia Nacional**: A otras cuentas en España (límite 30€ por operación)
- **Transferencia Internacional**: Envíos al extranjero
- **Formulario Completo**: Selección de cuenta origen, beneficiario, importe y concepto
- **Resumen de Confirmación**: Verificación antes de ejecutar

### 8. Estadísticas de Gastos

- **Visualización de Datos**: Gráficos y análisis de gastos
- **Categorización**: Seguimiento por categorías de gasto
- **Reportes**: Informes de actividad financiera

### 9. Perfil de Usuario

- **Información Personal**: Gestión de datos del usuario
- **Avatar Personalizable**: Imagen de perfil customizable
- **Configuración de Cuenta**: Preferencias y ajustes
- **Cerrar Sesión**: Salida segura de la aplicación

### 10. SabaDingo - Educación Financiera

Módulo educativo gamificado para aprender sobre finanzas (en desarrollo)

### 11. Información Legal y Transparencia

- **Política de Privacidad**: Información detallada sobre recopilación y uso de datos
- **Términos y Condiciones**: Reglas de uso, límites y responsabilidades
- **Acerca de Sabadell First**: Misión, características y compromiso educativo
- **Contactos**: Soporte, educación y atención a padres

## Características Técnicas

### Control Parental

- **Límite de gasto diario**: 50€
- **Límite por transferencia**: 30€
- **Máximo mensual en compras online**: 100€
- **Supervisión obligatoria**: Los padres tienen acceso completo a la información
- **Alertas en tiempo real**: Notificaciones de todas las transacciones

### Funcionalidades Planeadas

**Para Padres/Tutores**:
- Límites de gasto personalizables (semanal/mensual)
- Resumen de gastos automático
- Recomendaciones ajustadas según patrones de gasto

**Sistema de Amigos**:
- Agregar amigos sin acceso a saldos
- Ranking de ahorradores
- Misiones conjuntas de gasto responsable

**Cashback Inteligente**:
- Porcentaje automático a hucha de ahorro
- 5% cashback en hábitos saludables
- Proyecciones de ahorro ("Si hicieras esto cada semana, ahorrarías 20€ al año")

### Seguridad

- Encriptación de datos
- Autenticación segura
- Tecnología bancaria profesional
- Garantía de depósitos hasta 100.000€ (Fondo de Garantía de Depósitos)
- Supervisión del Banco de España

## Despliegue

### Opción 1: Desarrollo Local Simple

Abre directamente `index.html` en tu navegador, o usa un servidor local:

```bash
# Con Python 3
python -m http.server 8080

# Con Python 2
python -m SimpleHTTPServer 8080

# Con Node.js (http-server)
npx http-server -p 8080

# Con Node.js (serve)
npx serve -p 8080

# Con PHP
php -S localhost:8080
```

Accede a: `http://localhost:8080`

### Opción 2: Docker

```bash
# Construir la imagen
docker build -t sabadell-first .

# Ejecutar el contenedor
docker run -d -p 8080:80 --name sabadell-first sabadell-first
```

### Opción 3: Podman

```bash
# Crear el pod
podman pod create --name sabadell-first -p 8080:80

# Construir la imagen
podman build -t sabadell-first .

# Ejecutar el contenedor
podman run -d --pod sabadell-first --name sabadell-first-container sabadell-first
```

### Opción 4: GitHub Pages

El proyecto está configurado para desplegarse en GitHub Pages:

1. Push a la rama `main`
2. Accede a: `https://jas77-ua.github.io/IBHF25.github.io/`
